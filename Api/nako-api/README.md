# Nako Backend
## Requirements
- 1x non retarded homosapient
- 1x Computer
- Python (>=3.6)
## Dependencies
- Flask (>=0.1)
- Flask-RESTful
- Flask_cors
- qrcode
### Install
```
$ pip3 install -c requirement.txt
# pls see requirement.txt for latest requirement
```
### or install maunally
```
$ pip3 install flask-restful
$ pip3 install flask
$ pip3 install flask_cors
$ pip3 install qrcode
```
## How 2 run
```
$ python3 nakoRest.py
$ python3 ws.py
```
